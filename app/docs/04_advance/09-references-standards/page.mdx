- **Level 1**: Resource-based URLs (`/users`, `/orders`)
- **Level 2**: HTTP verbs and status codes (`GET`, `POST`, `PUT`, `DELETE`)
- **Level 3**: Hypermedia controls (HATEOAS) for API discoverability
