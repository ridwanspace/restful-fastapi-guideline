## What is Anthropic?

Anthropic is an AI safety company that develops Claude, a helpful, harmless, and honest AI assistant. The Python SDK provides a simple and intuitive way to integrate Claude into your applications.

## Quick Start

```python
from anthropic import Anthropic

client = Anthropic(api_key="your-api-key")

response = client.messages.create(
    model="claude-3-opus-20240229",
    max_tokens=100,
    messages=[
        {"role": "user", "content": "Hello, Claude!"}
    ]
)

print(response.content[0].text)
```
